## ■ 依頼内容
Sea&Bunny「DAYLOCK」に関連する異常挙動について、現場調査の依頼を発行します。  
本依頼は、調査員による観測・検証の結果をもって進行します。

## ■ 調査開始
開始時刻：07:00

## ■ 調査範囲
- Sea&beluga（1F カーディーラー）
- Sea&Bunny（飲食店）

## ■ 調査目的
異常の“起点”と“条件”を特定してください。  
いつ・誰が・どんな操作や出来事をきっかけにループが始まり、何を満たせば止まるのかを、  
観測と検証で突き止めること。

## ■ 進行の要点
- ロケーションを移動し、「話す／調べる」を用いて調査を進めてください
- 入力したワードにより、移動先や情報が開示される場合があります
- 移動／調べる／話す は、それぞれ時間が進行します（各 +1時間）
- 異常により、記憶の欠如が発生する可能性があります
